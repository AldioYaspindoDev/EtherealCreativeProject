# 🌟 Ethereal Creative – Fashion E-Commerce Website

Ethereal Creative adalah website e-commerce yang dikembangkan untuk perusahaan **Ethereal**, sebuah brand fashion yang bergerak di bidang penjualan **pakaian dan jeans** yang berlokasi di Kota Padang.  
Project ini dibangun untuk membantu proses penjualan secara digital, mulai dari katalog produk, manajemen transaksi, hingga sistem autentikasi pengguna.

---

## 🚀 Tech Stack

### **Frontend**
- Next.js  
- React.js  
- TailwindCSS  
- Axios  

### **Backend**
- Node.js  
- Express.js  
- REST API  
- JWT Authentication  

### **Database**
- MongoDB (Mongoose)

---

## ✅ Fitur Utama

- **📦 Product Catalog**  
  Menampilkan kumpulan produk (foto, harga, deskripsi).

- **🛒 Shopping Cart**  
  Pengguna dapat menambahkan produk ke keranjang sebelum checkout.

- **🔐 User Authentication**  
  Register, login, dan proteksi halaman menggunakan JWT.

- **🛠 Admin Dashboard**  
  - CRUD produk  
  - Manajemen order  
  - Manajemen pengguna

- **📱 Fully Responsive**  
  UI ramah untuk pengguna mobile, tablet, hingga desktop.

- **⚙ Modern REST API**  
  Membangun komunikasi antara frontend dan backend yang terstruktur.
